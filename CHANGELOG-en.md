- **1.6** New features :
    - Support for the Google Gemini API for translation (`--use_gemini`)
    - Update of the 2026 default models :
        - OpenAI: `gpt-5` (quality), `gpt-5-mini` (eco)
        - Claude: `claude-sonnet-4-5` (quality), `claude-haiku-4-5` (eco)
        - Gemini: `gemini-3-pro-preview` (quality), `gemini-3-flash-preview` (eco)
    - Economy mode (`--eco`) to use faster and less expensive models
    - Single-file translation (`--file`) without traversing a directory
    - New simplified naming pattern: `{base}-{lang}.md`
    - Option `--include_model` to keep the old format with the model name
    - Support for unlisted models with default token limit (128k)
    - README translated into 14 languages
- **1.5** Improvements :
    - **Update of API keys and default models:**
        - **OpenAI:** Updated `DEFAULT_MODEL_OPENAI` to `"gpt-4o"`.
        - **Mistral AI:** Updated `DEFAULT_MODEL_MISTRAL` to `"mistral-large-latest"`.
        - **Anthropic Claude:** Added `DEFAULT_ANTHROPIC_API_KEY` and updated `DEFAULT_MODEL_CLAUDE` to `"claude-3-5-sonnet-20240620"`.
    - **Optimization of translation prompts:**
        - Prompts for direct translations and translation notes have been enhanced for better clarity and efficiency, including detailed instructions on preserving metadata and specific formatting elements.
    - **Code refactoring:**
        - Replaced `MistralClient` with the `Mistral` class for initializing the Mistral AI client.
        - Reorganized imports for better readability and maintainability.
        - Improved text segmentation and handling of code blocks to preserve the original formatting during translation.
    - **Output file handling:**
        - Inversion of the model and the language in the output file names (for example, `f"{base}-{args.target_lang}-{args.model}.md"`), thereby making it easier to organize and search the translations.
    - **Miscellaneous improvements:**
        - Code cleanup by removing unnecessary blank lines.
        - Minor adjustments to improve the script's structure and readability.
