### سجل التغييرات

- **1.0** الإصدار الأول - دعم API OpenAI
- **1.1** إضافة دعم API Mistral IA
- **1.2** إصلاح سجل التغييرات
- **1.3** تحسينات وميزات جديدة:
    - تحسين إدارة كتل الكود
    - تحسين إدارة ملفات المخرجات
    - تحسين اكتشاف الملفات الموجودة
    - خيار `--force` لإجبار الترجمة
    - عكس النموذج واللغة في اسم ملف المخرجات
- **1.4** مستجدات:
    - دعم API Claude d'Anthropic للترجمة
    - تحسين الاستفسارات لتحقيق وضوح وكفاءة أكبر
    - تعديلات طفيفة لتحسين صيانة الكود
- **1.5** تحسينات:
    - **تحديث مفاتيح API والنماذج الافتراضية:**
        - **OpenAI:** تحديث `DEFAULT_MODEL_OPENAI` إلى `"gpt-4o"`.
        - **Mistral AI:** تحديث `DEFAULT_MODEL_MISTRAL` إلى `"mistral-large-latest"`.
        - **Claude d'Anthropic:** إضافة `DEFAULT_ANTHROPIC_API_KEY` وتحديث `DEFAULT_MODEL_CLAUDE` إلى `"claude-3-5-sonnet-20240620"`.
    - **تحسين استفسارات الترجمة:**
        - تم إثراء الاستفسارات للترجمات المباشرة والملاحظات الترجمة لتحقيق وضوح وكفاءة أفضل، بما في ذلك تعليمات مفصلة للحفاظ على البيانات الوصفية وعناصر التنسيق الخاصة.
    - **إعادة هيكلة الكود:**
        - استبدال `MistralClient` بفئة `Mistral` لتهيئة عميل Mistral AI.
        - إعادة تنظيم الاستيرادات لتحسين قابلية قراءة وصيانة الكود.
        - تحسين تقسيم النصوص وإدارة كتل الكود للحفاظ على التنسيق الأصلي أثناء الترجمة.
    - **إدارة ملفات المخرجات:**
        - عكس النموذج واللغة في اسم ملفات المخرجات (على سبيل المثال، `f"{base}-{args.target_lang}-{args.model}.md"`)، مما يسهل تنظيم وبحث الترجمات.
    - **تحسينات متفرقة:**
        - تنظيف الكود بإزالة الأسطر الفارغة غير الضرورية.
        - تعديلات طفيفة لتحسين هيكلية وقابلية قراءة النص البرمجي.
