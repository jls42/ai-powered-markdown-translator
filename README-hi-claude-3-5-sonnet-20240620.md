# OpenAI, Mistral AI और Anthropic के Claude द्वारा संचालित AI-पावर्ड मार्कडाउन अनुवादक

यह प्रोजेक्ट एक उन्नत पायथन स्क्रिप्ट है जो OpenAI, Mistral AI या Anthropic के Claude API का उपयोग करके मार्कडाउन फाइलों को एक स्रोत भाषा से लक्ष्य भाषा में अनुवाद करता है। यह लचीला और उपयोग में आसान होने के लिए डिज़ाइन किया गया है, जो अतिरिक्त विकल्प प्रदान करता है जैसे अनुवाद नोट जोड़ना, बेहतर आउटपुट फाइल प्रबंधन, मौजूदा फाइलों का पता लगाना, और कई भाषाओं और अनुवाद मॉडल का समर्थन।

एक डेमो और विस्तृत व्याख्या के लिए, [jls42.org](https://jls42.org/) पर जाएं या अनुवादित संस्करण में: [jls42.org English](https://jls42.org/traductions_en/), [jls42.org Español](https://jls42.org/traductions_es/) और [jls42.org 中文](https://jls42.org/traductions_zh/)।

## मुख्य विशेषताएं

- **AI-संचालित अनुवाद**: OpenAI, Mistral AI या Anthropic के Claude के साथ अपने दस्तावेज़ों के अनुवाद के लिए नवीनतम AI तकनीकों का उपयोग करें।
- **बहुभाषी समर्थन**: विभिन्न भाषा मॉडल के समर्थन के साथ अपने दस्तावेज़ों का कई भाषाओं में अनुवाद करें।
- **स्मार्ट सेगमेंटेशन**: स्वचालित सेगमेंटेशन के माध्यम से लंबे पाठों का कुशलतापूर्वक प्रबंधन करें।
- **अनुवाद नोट**: पाठकों को उपयोग की गई प्रक्रिया के बारे में सूचित करने के लिए स्वचालित रूप से एक अनुवाद नोट जोड़ें।
- **बेहतर आउटपुट फाइल प्रबंधन**: अनुवाद शुरू करने से पहले जांचें कि क्या अनुवाद पहले से मौजूद है।
- **बेहतर मौजूदा फाइल पहचान**: मूल फाइल के आधार नाम और लक्ष्य भाषा से मेल खाने वाली फाइलों की खोज करें।
- **लचीला और विस्तारणीय**: कोड नई सुविधाओं को आसानी से जोड़ने की अनुमति देने के लिए संरचित है।

## आवश्यकताएं

- Python 3.6 या उच्चतर संस्करण।
- OpenAI, Mistral AI या Anthropic के Claude के लिए एक वैध API कुंजी।

## स्थापना

1. Git रिपॉजिटरी को क्लोन करें:
```
git clone https://gitlab.com/jls42/ai-powered-markdown-translator.git
```
2. आवश्यक निर्भरताओं को स्थापित करें:
```
pip install -r requirements.txt
```

## कॉन्फ़िगरेशन

आवश्यक API कुंजियों के लिए पर्यावरण चर सेट करके अपने वातावरण को कॉन्फ़िगर करें:

- **OpenAI**:
    ```
    export OPENAI_API_KEY='आपकी-openai-api-कुंजी'
    ```
- **Mistral AI**:
    ```
    export MISTRAL_API_KEY='आपकी-mistral-api-कुंजी'
    ```
- **Anthropic का Claude**:
    ```
    export ANTHROPIC_API_KEY='आपकी-anthropic-api-कुंजी'
    ```

## उपयोग

स्क्रिप्ट अनुवाद प्रक्रिया को अनुकूलित करने के लिए कई विकल्प प्रदान करती है:

### सामान्य विकल्प

- `--source_dir`: अनुवाद के लिए मार्कडाउन फाइलों वाला निर्देशिका।
- `--target_dir`: अनुवादित फाइलों के लिए आउटपुट निर्देशिका।
- `--model`: उपयोग करने के लिए GPT अनुवाद मॉडल। डिफ़ॉल्ट मॉडल चयनित API पर निर्भर करता है।
- `--source_lang`: दस्तावेज़ों की स्रोत भाषा। विशेष रूप से अनुवाद नोट्स जोड़ने के लिए महत्वपूर्ण।
- `--target_lang`: अनुवाद के लिए लक्ष्य भाषा। डिफ़ॉल्ट रूप से, यह अंग्रेजी है।
- `--force`: फाइल के लिए पहले से अनुवाद मौजूद होने पर भी अनुवाद को बल देना।

### विशिष्ट API विकल्प

- `--use_mistral`: अनुवाद के लिए Mistral AI API का उपयोग करें।
- `--use_claude`: अनुवाद के लिए Anthropic के Claude API का उपयोग करें।
- `--add_translation_note`: अनुवादित सामग्री में एक अनुवाद नोट जोड़ें, जिसमें उपयोग किए गए विधि और उपकरणों को निर्दिष्ट किया गया हो।

### उपयोग के उदाहरण

- फ्रेंच से अंग्रेजी में OpenAI के साथ अनुवाद करें, एक अनुवाद नोट जोड़ें:
    ```
    python translate.py --source_dir 'content/fr' --target_dir 'content/en' --add_translation_note --source_lang 'fr'
    ```
- फ्रेंच से स्पेनिश में Mistral AI के साथ अनुवाद करें, अनुवाद नोट के बिना:
    ```
    python translate.py --use_mistral --source_dir 'content/fr' --target_dir 'content/es' --target_lang 'es'
    ```
