# ओपनएआई, मिस्ट्रल एआई और क्लॉड ऑफ़ आन्थ्रोपिक द्वारा संचालित मार्कडाउन अनुवादक एआई-पावर्ड

यह परियोजना एक उन्नत पायथन स्क्रिप्ट है जो ओपनएआई, मिस्ट्रल एआई या क्लॉड ऑफ़ आन्थ्रोपिक एपीआई का उपयोग करके मार्कडाउन फ़ाइलों को एक स्रोत भाषा से लक्ष्य भाषा में अनुवादित करती है। इसे लचीला और उपयोग में आसान बनाने के लिए डिज़ाइन किया गया है, अतिरिक्त विकल्पों के साथ जैसे अनुवाद नोट जोड़ना, आउटपुट फाइलों का उन्नत प्रबंधन, मौजूदा फ़ाइलों का पता लगाना, और कई भाषाओं और अनुवाद मॉडलों के समर्थन के साथ।

डेमो और विस्तृत स्पष्टीकरण के लिए, [jls42.org](https://jls42.org/) पर जाएं या अनुवादित संस्करणों में: [jls42.org English](https://jls42.org/traductions_en/), [jls42.org Español](https://jls42.org/traductions_es/) और [jls42.org 中文](https://jls42.org/traductions_zh/)।

## प्रमुख विशेषताएं

- **एआई-पावर्ड अनुवाद**: अपने दस्तावेज़ों के अनुवाद के लिए ओपनएआई, मिस्ट्रल एआई या क्लॉड ऑफ़ आन्थ्रोपिक के साथ नवीनतम एआई तकनीकों का उपयोग करें।
- **मल्टीलिंगुअल सपोर्ट**: विभिन्न भाषा मॉडलों के समर्थन के साथ अपने दस्तावेज़ों का कई भाषाओं में अनुवाद करें।
- **स्मार्ट सेगमेंटेशन**: स्वचालित सेगमेंटेशन के साथ लंबे ग्रंथों का कुशलतापूर्वक प्रबंधन करें।
- **अनुवाद नोट**: उपयोग किए गए प्रक्रिया के बारे में पाठकों को सूचित करने के लिए स्वचालित रूप से अनुवाद नोट जोड़ें।
- **आउटपुट फाइलों का उन्नत प्रबंधन**: अनुवाद शुरू करने से पहले देखें कि क्या कोई अनुवाद पहले से ही मौजूद है।
- **मौजूदा फ़ाइलों का उन्नत पता लगाना**: स्रोत फ़ाइल के मूल नाम और लक्ष्य भाषा से मेल खाने वाली फाइलों को खोजें।
- **लचीला और एक्स्टेंसिबल**: नई सुविधाओं को जोड़ने में आसानी की अनुमति देने के लिए कोड संरचित है।

## विन्यास

आवश्यक एपीआई कुंजियों के लिए पर्यावरण वेरिएबल्स सेट करके अपने पर्यावरण को कॉन्फ़िगर करें:

    ```
    export ANTHROPIC_API_KEY='आपकी-आन्थ्रोपिक-एपीआई-कुंजी'
    ```

## उपयोग

स्क्रिप्ट अनुवाद प्रक्रिया को वैयक्तिकृत करने के लिए कई विकल्प प्रदान करती है:

### सामान्य विकल्प

- `--source_dir` : अनुवाद के लिए मार्कडाउन फाइलों वाला डिरेक्टरी।
- `--target_dir` : अनुवादित फाइलों के लिए आउटपुट डिरेक्टरी।
- `--model` : उपयोग करने के लिए जीपीटी अनुवाद मॉडल। डिफ़ॉल्ट मॉडल चुनी गई एपीआई पर निर्भर करता है।
- `--source_lang` : दस्तावेज़ों की स्रोत भाषा। विशेष रूप से अनुवाद नोट्स जोड़ने के लिए महत्वपूर्ण है।
- `--target_lang` : अनुवाद के लिए लक्ष्य भाषा। डिफ़ॉल्ट रूप से, यह अंग्रेजी है।
- `--force` : उस स्थिति में भी अनुवाद करने के लिए मजबूर करना यदि किसी फ़ाइल के लिए पहले से ही एक अनुवाद मौजूद है।

### विशिष्ट एपीआई विकल्प

- `--use_mistral` : अनुवाद के लिए मिस्ट्रल एआई एपीआई का उपयोग करें।
- `--use_claude` : अनुवाद के लिए क्लॉड ऑफ़ आन्थ्रोपिक एपीआई का उपयोग करें।
- `--add_translation_note` : उपयोग की गई विधि और उपकरण को निर्दिष्ट करते हुए अनुवादित सामग्री में अनुवाद नोट जोड़ें।

### उपयोग के उदाहरण

- ओपनएआई के साथ अनुवाद नोट जोड़ते हुए फ्रेंच से अंग्रेजी में अनुवाद करना:
    ```
    python translate.py --source_dir 'content/fr' --target_dir 'content/en' --add_translation_note --source_lang 'fr'
    ```
- मिस्ट्रल एआई के साथ अनुवाद नोट के बिना फ्रेंच से स्पैनिश में अनुवाद करना:
    ```
    python translate.py --use_mistral --source_dir 'content/fr' --target_dir 'content/es' --target_lang 'es'
    ```
