- **1.6** المستجدات :
    - دعم API Google Gemini للترجمة (`--use_gemini`)
    - تحديث النماذج الافتراضية لعام 2026 :
        - OpenAI : `gpt-5` (جودة), `gpt-5-mini` (اقتصادي)
        - Claude : `claude-sonnet-4-5` (جودة), `claude-haiku-4-5` (اقتصادي)
        - Gemini : `gemini-3-pro-preview` (جودة), `gemini-3-flash-preview` (اقتصادي)
    - الوضع الاقتصادي (`--eco`) لاستخدام نماذج أسرع وأرخص
    - ترجمة ملف واحد (`--file`) دون استعراض مجلد
    - نمط تسمية مبسّط جديد : `{base}-{lang}.md`
    - الخيار `--include_model` للاحتفاظ بالتنسيق القديم مع اسم النموذج
    - دعم النماذج غير المدرجة مع حد افتراضي للرموز (128k)
    - README مترجم إلى 14 لغة
- **1.5** تحسينات :
    - تحديث مفاتيح API والنماذج الافتراضية :
        - OpenAI : تم التحديث من `DEFAULT_MODEL_OPENAI` إلى `"gpt-4o"`.
        - Mistral AI : تم التحديث من `DEFAULT_MODEL_MISTRAL` إلى `"mistral-large-latest"`.
        - Claude من Anthropic : إضافة `DEFAULT_ANTHROPIC_API_KEY` وتحديث `DEFAULT_MODEL_CLAUDE` إلى `"claude-3-5-sonnet-20240620"`.
    - تحسين مطالبات الترجمة :
        - تم إثراء المطالبات للترجمات المباشرة وملاحظات الترجمة لتحسين الوضوح والكفاءة، بما في ذلك تعليمات مفصّلة حول الحفاظ على البيانات الوصفية وعناصر التنسيق المحددة.
    - إعادة هيكلة الكود :
        - استبدال `MistralClient` بالفئة `Mistral` لتهيئة عميل Mistral AI.
        - إعادة تنظيم الاستيرادات لتحسين قابلية القراءة والصيانة.
        - تحسين تقسيم النصوص وإدارة كتل الشيفرة للحفاظ على التنسيق الأصلي أثناء الترجمة.
    - إدارة ملفات الإخراج :
        - عكس ترتيب النموذج واللغة في اسم ملفات الإخراج (على سبيل المثال، `f"{base}-{args.target_lang}-{args.model}.md"`)، مما يسهل تنظيم والبحث عن الترجمات.
    - تحسينات متنوعة :
        - تنظيف الكود بحذف الأسطر الفارغة غير الضرورية.
        - تعديلات بسيطة لتحسين هيكلية وقابلية قراءة السكربت.
