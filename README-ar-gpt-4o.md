# مترجم Markdown المدعوم بالذكاء الاصطناعي مع OpenAI و Mistral AI وكلود من Anthropic

هذا المشروع هو برنامج نصي متقدم بلغة بايثون يستخدم واجهات برمجة التطبيقات الخاصة بـ OpenAI وMistral AI أو كلود من Anthropic لترجمة ملفات Markdown من لغة المصدر إلى اللغة المستهدفة. تم تصميمه ليكون مرنًا وسهل الاستخدام، مع تقديم خيارات إضافية مثل إضافة ملاحظة ترجمة، إدارة محسنة لملفات الإخراج، اكتشاف الملفات الموجودة، ودعم للعديد من اللغات ونماذج الترجمة.

للحصول على عرض توضيحي وشرح مفصل، قم بزيارة [jls42.org](https://jls42.org/) أو النسخة المترجمة: [jls42.org English](https://jls42.org/traductions_en/) و[jls42.org Español](https://jls42.org/traductions_es/) و[jls42.org 中文](https://jls42.org/traductions_zh/).

## الميزات الرئيسية

- **ترجمة مدعومة بالذكاء الاصطناعي**: استخدم أحدث تقنيات الذكاء الاصطناعي لترجمة مستنداتك باستخدام OpenAI وMistral AI أو كلود من Anthropic.
- **دعم متعدد اللغات**: ترجم مستنداتك إلى عدة لغات مع دعم لمختلف نماذج اللغة.
- **تقسيم ذكي**: إدارة فعالة للنصوص الطويلة بفضل التقسيم الآلي.
- **ملاحظة ترجمة**: أضف تلقائيًا ملاحظة ترجمة لإبلاغ القراء عن العملية المستخدمة.
- **إدارة محسنة لملفات الإخراج**: تحقق مما إذا كانت هناك ترجمة موجودة بالفعل قبل بدء الترجمة.
- **اكتشاف محسّن للملفات الموجودة**: ابحث عن الملفات المطابقة لاسم الملف الأصلي واللغة المستهدفة.
- **مرن وقابل للتطوير**: الكود مصمم لتسهيل إضافة ميزات جديدة.

## المتطلبات

- بايثون 3.6 أو إصدار لاحق.
- مفتاح API صالح لـ OpenAI أو Mistral AI أو كلود من Anthropic.

## التثبيت

1. استنساخ مستودع Git:
```
git clone https://gitlab.com/jls42/ai-powered-markdown-translator.git
```
2. تثبيت التبعيات اللازمة:
```
pip install -r requirements.txt
```

## الاستخدام

يوفر البرنامج النصي عدة خيارات لتخصيص عملية الترجمة:

### الخيارات العامة

- `--source_dir`: الدليل الذي يحتوي على ملفات Markdown المراد ترجمتها.
- `--target_dir`: دليل الإخراج للملفات المترجمة.
- `--model`: نموذج ترجمة GPT المستخدم. النموذج الافتراضي يعتمد على واجهة برمجة التطبيقات المختارة.
- `--source_lang`: لغة مستندات المصدر. مهم بشكل خاص لإضافة الملاحظات الترجمة.
- `--target_lang`: اللغة المستهدفة للترجمة. الافتراضي هو اللغة الإنجليزية.
- `--force`: فرض الترجمة حتى إذا كانت هناك ترجمة موجودة بالفعل للملف.

### خيارات API الخاصة

- `--use_mistral`: استخدام واجهة برمجة تطبيقات Mistral AI للترجمة.
- `--use_claude`: استخدام واجهة برمجة تطبيقات كلود من Anthropic للترجمة.
- `--add_translation_note`: إضافة ملاحظة ترجمة إلى المحتوى المترجم، محددًا الطريقة والأدوات المستخدمة.

### أمثلة الاستخدام

- ترجمة من الفرنسية إلى الإنجليزية باستخدام OpenAI، مع إضافة ملاحظة ترجمة:
    ```
    python translate.py --source_dir 'content/fr' --target_dir 'content/en' --add_translation_note --source_lang 'fr'
    ```
- ترجمة من الفرنسية إلى الإسبانية باستخدام Mistral AI، بدون ملاحظة ترجمة:
    ```
    python translate.py --use_mistral --source_dir 'content/fr' --target_dir 'content/es' --target_lang 'es'
    ```
